# Activity Description 
I created a portfolio website to facilitate contact with potential employers. I learned how to host a site with github pages and how web design
works in HTML/CSS. 

# Technical/Professional Decisions
After researching resume sites and templates, I decided I wanted to challenge myself (both to learn more HTML/CSS and satisfy the CYA guidelines)
and not rely on a pre-existing templates.
I used html5 boilerplate and nodejs as they were included in many tutorial resources I found, otherwise minimal plugins were used. 

# Contributions
This project was done individually. GeeksForGeeks and other free coding resources were used when I ran into bugs or wanted
to find out how to format my site a specific way using css. This project has been something I've been working on in my freetime 
throughout the latter half of semester, as a portfolio website has been encouraged in a few of my courses.

# Quality Assessment
I really like how my site turned out, it looks more unique than a lot of the templates I found online and I think it helps
me stick out. It effeciently routes a viewer to all the information they could need to know my skills.
In the future, I want to add a carousel or display of my projects and improve my cv page, I also want to improve the repo 
organization. 

# Career Alignment
This project helped me to learn more about front-end web development, as typically in CS courses thus far, I work mostly with 
non-graphical, object-oriented programming. This project helped introduce me to javascript, HTML, and CSS. This also will be a good
resource for online job applications.
